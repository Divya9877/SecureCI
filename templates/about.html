About SecureCI

SecureCI is an intelligent Software Supply Chain Security Scanner designed to identify vulnerable dependencies before software deployment.

Project Goal

The main goal of SecureCI is to improve DevSecOps security by automatically scanning third-party dependencies and preventing risky deployments in CI/CD pipelines.

Key Features

  • Dependency Vulnerability Scanning
  • Risk Score Calculation
  • Policy-Based Deployment Blocking
  • SBOM Generation
  • Prometheus + Grafana Monitoring
  • MongoDB Scan History
  • Postman API Testing
  • CI/CD Integration

Technologies Used

  • Flask
  • Python
  • MongoDB
  • Docker
  • GitHub Actions
  • Prometheus
  • Grafana
  • Ansible
  • HTML / CSS

Workflow

Users upload dependency files or manually enter dependencies. SecureCI scans packages, detects vulnerable libraries, calculates risk, applies deployment policies, generates reports, stores scan history in MongoDB, and exposes monitoring metrics to Grafana dashboards.
